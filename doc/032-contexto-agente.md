# Contexto Disponible para los Agentes

## Información del Expediente

El agente tiene acceso a **toda la información del Expediente** sobre el que está trabajando:

- Datos del expediente
- Documentos asociados
- Historial de acciones
- Metadatos relevantes

## Alcance del Contexto

El contexto está limitado al expediente específico que está siendo procesado. El agente no tiene acceso a información de otros expedientes.
