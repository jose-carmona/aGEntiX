# Zettelkasten

## Principios Generales

- Las notas están escritas en Markdown.
- Cada nota representa una idea, un concepto, un requisito, etc.
- Cada nota se compone normalmente de:
  - Un título.
  - El texto de la nota.
  - Relaciones con otras notas.
- Existirán notas índice.

## Formato de Enlaces

Los enlaces entre notas utilizan el formato **markdown estándar**:

```markdown
[Título descriptivo de la nota](nombre-archivo.md)
```

**Ejemplos:**
- `[Componentes del sistema](002-componentes-gex.md)`
- `[Permisos del agente](050-permisos-agente.md)`

**No usar** el formato wiki `[[nombre-archivo]]`, ya que el formato markdown estándar ofrece mejor compatibilidad con visores y herramientas.
