# Requisitos de Auditoría y Logging

## Obligación de Logging

El agente **debe dejar log de todos los pasos** que realiza.

## Propósitos del Logging

1. **Depuración**: Identificar y corregir problemas en el funcionamiento del agente
2. **Comprobación**: Verificar que lo realizado por el agente es correcto
3. **Auditoría**: Mantener trazabilidad de las acciones automatizadas
4. **Cumplimiento**: Requisito para entornos administrativos regulados

## Importancia

El logging es fundamental para mantener la confianza en el sistema y cumplir con los requisitos de transparencia de la administración pública.
