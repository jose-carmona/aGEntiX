# Estructura de una Tarea BPMN

## Composición

Una Tarea se descompone en **Acciones**.

## Encadenamiento de Tareas

### Tareas Siguientes Condicionales
Las tareas pueden tener múltiples tareas siguientes en función de condiciones evaluadas.

### Selección Manual
En algunos casos, la tarea siguiente es seleccionada por un humano.

### Tareas con Timeout
Las tareas pueden tener un **timeout** que conduce a otra tarea (normalmente, de comprobación).
