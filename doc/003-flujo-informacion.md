# Flujo de Información

El flujo de información en el sistema sigue el siguiente esquema:

```
Ciudadano → Sede Electrónica → GEX → Aviso/Notificación Electrónica → Ciudadano → Realimentación
```

## Elementos del Flujo

1. **Ciudadano**: Origen de la solicitud/trámite
2. **Sede Electrónica**: Portal de entrada para trámites
3. **GEX**: Sistema central de procesamiento
4. **Notificación Electrónica**: Sistema de comunicación con el ciudadano
5. **Realimentación**: Respuesta o acción del ciudadano
