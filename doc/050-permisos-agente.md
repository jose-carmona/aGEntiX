# Sistema de Permisos para Agentes IA

## Permisos de Lectura

**¿Qué información puede ver un agente?**

Toda la información del **Expediente sobre el que se está trabajando**.

## Permisos de Escritura

Los permisos de escritura están **limitados por la configuración** del agente.

No todos los agentes tienen los mismos permisos de escritura. Estos se configuran según las necesidades y el nivel de confianza de cada tipo de agente.

## Principio de Mínimo Privilegio

Los agentes deben tener únicamente los permisos necesarios para realizar su función específica.
