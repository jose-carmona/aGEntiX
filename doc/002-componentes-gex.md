# Componentes de GEX

GEX está compuesto por tres partes principales:

## 1. Gestor Documental
Sistema de almacenamiento y gestión de documentos.

## 2. Gestor de Expedientes
Sistema que gestiona los expedientes (casos) administrativos.

## 3. Motor BPMN
Motor de automatización de workflows que permite la tramitación automatizada de procesos.
